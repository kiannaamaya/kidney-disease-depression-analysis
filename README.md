# The Association between Kidney Disease and Depression Among US Adults in 2021

Welcome to the repository for my project! This project investigates the relationship between kidney disease and depression in the US population, using data from the Behavioral Risk Factor Surveillance System (BRFSS) 2021.

## Dataset

The dataset used in this project is sourced from the 2021 BRFSS, which is a United States health survey that looks at behavioral risk factors. 

The 2021 BRFSS data, which this project is based upon, can be accessed from [CDC's website](https://www.cdc.gov/brfss/annual_data/annual_2021.html). 

Researchers interested in this project should download the data directly from the official website.

## Project Description

This project aims to:

- Explore the association between kidney disease and depression among adults in the United States.
- Provide insights for future research, emphasizing the importance of mental health care in managing kidney disease.
